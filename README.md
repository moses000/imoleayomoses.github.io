## Overview
This portfolio is a dynamic, responsive single-page application designed to present my professional journey and technical capabilities. It includes interactive elements like a projects carousel, real-time performance monitor, and particle animations, reflecting my passion for innovative software solutions.

## Features
- **Hero Section**: Animated introduction with a tagline: "Architecting Scalable AI Solutions for a Hyper-Connected Future."
- **About Me**: Professional summary of my 6+ years of experience, core competencies (software development, AI/automation, DevOps, quantum computing), and career highlights.
- **Experience**: Timeline of my work history with companies like RGS World Wide Limited (Huawei) and DevFast Innovations.
- **Skills**: Progress bars for languages (Python, JavaScript, etc.), a tech cloud of tools/frameworks, and a live CPU performance monitor.
- **Projects**: Responsive carousel showcasing 6+ projects with demo links and tech stacks (e.g., Server Monitoring Dashboard, Pikia Market).
- **Awards**: Flip-card display of accolades and recommendations.
- **Contact**: Form and direct contact info for collaboration opportunities.
- **Interactive Elements**: Theme toggle (dark/light), particle backgrounds, GSAP animations, and touch-enabled carousel.

## Technologies
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Libraries/Frameworks**: GSAP (animations), Particles.js (background effects)
- **Tools**: Git, GitHub Actions (for potential CI/CD)
- **Dependencies**: Font Awesome (icons), Google Fonts (Poppins, Inter)
- **Styling**: Custom CSS with responsive design and dark/light mode support

## Installation
To run this portfolio locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/[your-username]/[repo-name].git
   ```
2. Navigate to the project directory:
   ```bash
   cd [repo-name]
   ```
3. Open `index.html` in a browser (no server required for static files):
   ```bash
   open index.html
   ```
   *Note*: For full functionality (e.g., form submission), host it on a local server like `live-server` or deploy it.

## Usage
- Browse sections via the sticky navigation bar.
- Toggle between dark and light modes using the theme button.
- Interact with the projects carousel (click arrows, swipe on mobile).
- View real-time CPU impact in the Skills section’s performance monitor.
- Contact me through the form or linked details.
